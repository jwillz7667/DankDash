# @dankdash/portal

Vendor web portal — Next.js 15, App Router, Tailwind + shadcn/ui, deployed to
Vercel. The full Information Architecture and screen list live in
`docs/spec/DankDash-Technical-Spec.md` §6.

This package is intentionally a stub during Phase 0. It is scaffolded in
**Phase 13 — Vendor Portal: Auth & Shell** (see
`docs/CLAUDE-CODE-PHASES.md`). The stub scripts exist so `pnpm typecheck`,
`pnpm lint`, `pnpm test`, and `pnpm build` succeed at the workspace level
during phases 0–12.
