# @dankdash/checkout-web

Consumer checkout web — Next.js 15. Required because Apple App Store
guideline 1.4.3 prohibits cannabis purchases inside iOS apps; the iOS
consumer app browses, but checkout opens a Safari web view pointed at
`app.dankdash.com` served by this package.

See `docs/spec/DankDash-Technical-Spec.md` §10.4 for the Apple workaround
in full.

This package is a stub during Phases 0–17. It is scaffolded in
**Phase 18 — iOS Consumer: Cart, Checkout, Tracking**.
